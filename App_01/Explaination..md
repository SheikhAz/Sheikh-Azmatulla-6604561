In this project, we work with basic React Native features such as View, Image, and Stack.

The View component is used to create layouts and to organize different elements on the screen.

In the Image component, we use the source property and assign it using require(). Inside single or double quotes, we provide the path (address) of the image file.

In Stack, we use screenOptions={{ }} to define and customize different screen settings and properties that are required for the navigation.
